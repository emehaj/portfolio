# Personal Portfolio

This is my personal portfolio website created to showcase my projects, skills, and experience.  
It serves as a central place where visitors can explore my work and learn more about me as a developer.

## Live Demo

[em-personalportfolio.vercel.app/](https://em-personalportfolio.vercel.app/)

## Features

-   Display of personal projects with preview images
-   Project details such as technologies and links
-   Responsive design for different screen sizes
-   Clean and minimal UI

## Technologies Used

-   **HTML** – Structure of the website
-   **CSS** – Styling and layout
-   **JavaScript** – Interactivity and dynamic content
-   **Supabase** – Used as a backend service to store:
    -   Project data
    -   Preview images for the projects
